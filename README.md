# Inventario Biblioteca con Streamlit y OCR

Esta aplicación permite escanear el texto debajo del código de barras usando la cámara del celular y actualizar un archivo Excel en tiempo real.

## Funcionalidades
- Marca en verde los códigos existentes.
- Agrega nuevos códigos y los marca en morado.
- Permite descargar el Excel actualizado.

## Cómo desplegar
1. Subir este proyecto a GitHub.
2. Ir a [Streamlit Cloud](https://streamlit.io/cloud).
3. Crear una nueva app y seleccionar este repositorio.
4. Elegir `app.py` como archivo principal.
5. ¡Listo! Accede a la URL pública desde tu celular.
